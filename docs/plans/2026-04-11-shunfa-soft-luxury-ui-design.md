# 顺发小程序 Soft Luxury Productivity UI 设计

**日期**: 2026-04-11

## 目标

把当前小程序从“带一点编辑部风格的功能界面”重构成更接近高级消费品的克制型体验。新的界面需要看起来更安静、更高级、更统一，同时不牺牲现有的核心路径：看热点、选题、生成、预览、发布、看连更、配提醒。

## 设计方向

本次设计采用 `Soft Luxury Productivity`：

- 视觉气质偏高级生活方式产品，而不是效率工具或资讯产品
- 通过暖奶白、雾灰、浅沙色、鼠尾草绿建立温和但精致的品牌感
- 页面强调节奏、留白和材质感，不靠重色块和强装饰来制造层次
- 交互反馈轻、慢、稳，避免工具化压迫感

## 视觉系统

### 配色

- 主背景：暖奶白，不用纯白
- 主文字：偏灰黑，不用纯黑
- 强调色：低饱和鼠尾草绿
- 辅助色：雾灰、浅沙色、极淡金属描边
- 危险态：收敛的暖红，不用亮红

### 材质与层次

- 一级卡：主内容承载面，最干净，留白最多
- 二级卡：信息说明与辅助内容，弱层次，轻描边
- 三级卡：轻提示与备注区，用浅沙色或淡雾绿色承接情绪
- 阴影非常轻，主要靠明度、留白和边框层次建立高级感

### 排版

- 标题体系收成 3 级，减少过多装饰性英文标签
- 正文、辅助信息、状态文字通过字号和灰度拉开差异
- 模块之间优先靠节奏感分层，而不是靠大量卡片和标签堆叠

## 页面级设计

### 首页

首页改造成“今日写作入口”，而不是任务看板。

- 顶部为克制的欢迎区，只保留当天状态和轻提示
- 核心区域只保留一个主行动：开始今日写作
- 连更、积分、等级等成长信息做成更安静的摘要区
- 今日完成状态用柔和的完成感表达，不做强“打卡”叙事

目标是让首页像一个安静但明确的开始动作，而不是后台面板。

### 选题页

选题页改造成“精选灵感卡片”。

- 热点卡片像被挑选过的高质量条目，不像资讯列表
- 标题更克制，摘要更短，来源和时间作为次级信息
- 选中状态用描边、底色和按钮层级变化表达，不用重色大块高亮
- 页面底部只保留一个确认动作，强调“今天只做一个决定”

### 预览页

预览页改造成“私人草稿样张”。

- 正文阅读区成为页面主角，增强可读性和静谧感
- 来源、摘要、原文链接放入安静的参考区，不抢正文
- 质量提示像编辑备注，而不是警告弹框
- 主要操作层次清晰：修改、继续、发布

### 我的页

我的页改造成“个人写作档案”。

- 弱化 dashboard 感
- 连更、累计发文、等级与提醒做成更平静的档案式摘要
- 避免大量数字卡片堆叠
- 更强调长期积累感，而不是 KPI 感

### 设置页

设置页偏系统设置页风格。

- 提醒、订阅消息、时间设置采用分组列表
- 控件和留白保持一致的高端消费品节奏
- 能原地完成的操作尽量不跳出

## 组件与交互规范

### 按钮

- 主按钮：深灰或鼠尾草绿，圆角足够大，但不过分软萌
- 次按钮：浅底、细描边、轻对比
- 链接型动作：尽量收敛为文字级动作，避免竞争主操作

### 卡片

- 全局统一圆角、描边、留白与阴影体系
- 不同页面不再各自定义完全不同的卡片语法
- 通过一级、二级、三级卡片系统建立全局一致性

### 状态表达

- 状态切换优先依赖明度、描边、阴影变化
- 避免荧光色、夸张标签和强对比 stamp 风格

### 动效

- 页面进入：轻微上浮 + 淡入
- 卡片按下：极轻微缩放
- 状态切换：150-220ms，尽量只动 opacity 和 transform
- 不做跳跃、弹簧、夸张渐变或戏剧化切场

## 明确要移除或收敛的内容

- 收敛当前页面中偏“编辑部”或“晨报”叙事的英文概念标签
- 减少 emoji 和过强的人设口号
- 把当前偏纸感编辑风的配色切换到更安静的雾面高级感
- 统一 tabBar、按钮、描边、状态标签和信息密度

## 验收标准

- 首屏一眼看上去不像模板化小程序
- 首页、选题页、预览页、我的页属于同一品牌体系
- 信息层次比现在更清楚，但视觉负担更轻
- 不影响现有核心路径和交互效率
- 页面在微信开发者工具中表现稳定，无明显布局断裂
